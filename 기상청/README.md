# 기상청 데이터

기상청에서 수집한 대전 지역의 지상 관측·기상특보 데이터, 해양 관측 데이터, 단기예보 API 참고 자료입니다. 해양 부이 자료는 대전 지역이 아닌 해상 관측 지점의 데이터입니다.

| 파일 | 포함 정보 |
| --- | --- |
| `ASOS시간단위(대전).zip` | 대전 종관기상관측장비(ASOS) 시간 단위 자료입니다. 2020~2025년 자료가 연도별 압축 파일로 들어 있으며, 기온·강수량·풍속/풍향·습도·기압·일조·일사·적설·운량·시정·지면/지중 온도 등을 포함합니다. |
| `AWS시간단위(대전).zip` | 대전 지역 자동기상관측장비(AWS) 지점의 시간 단위 자료입니다. 2020~2025년 자료가 지점·연도별로 들어 있으며, 기온·풍향/풍속·강수량·현지/해면기압·습도·일사·일조를 포함합니다. |
| `기상특보(대전).csv` | 2024-01-07~2026-07-14 대전·세종·충청남도 권역의 기상특보 이력입니다. 발표시간, 대상 지역, 발효시각 및 특보 발표·해제 내용을 기록합니다. 파일 인코딩은 CP949입니다. |
| `해양기상부이1.zip`~`해양기상부이3.zip` | 해양기상부이 지점별 시간 단위 자료입니다. 2024~2025년 자료가 분할 저장돼 있으며, 풍속/풍향·돌풍·기압·습도·기온·수온·최대/유의/평균 파고·파주기·파향을 포함합니다. |
| `단기예보_요청_응답.png` | 기상청 단기예보 API의 요청·응답 필드 참고 이미지입니다. 요청에는 기준일시와 예보 격자 좌표를, 응답에는 발표일시·예보 좌표·자료구분코드·실황값 등을 사용합니다. |

압축 파일은 내부에 다시 압축된 연도별 또는 지점별 CSV 자료를 포함합니다.
